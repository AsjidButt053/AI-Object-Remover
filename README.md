# 🖼️ AI Object Remover

An **AI-powered Object Remover** developed in **Python** as a **Semester 5 Software Engineering project**. The application enables users to remove unwanted objects from images by selecting them through drag-and-select or text-based object identification. It uses Artificial Intelligence and Computer Vision techniques to generate clean, edited images while preserving the surrounding background.

---

## 📖 Project Overview

The AI Object Remover is designed to simplify image editing by allowing users to remove unwanted objects from photographs. Users can either drag over the object they want to remove or specify the object, and the system automatically reconstructs the image to produce a natural-looking result.

This project demonstrates the practical use of **Artificial Intelligence**, **Computer Vision**, and **Image Processing** techniques in Python.
